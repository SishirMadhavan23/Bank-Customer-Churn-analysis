# User Manual

## Bank Customer Churn Analysis

### Overview
This application predicts customer churn and provides analytics for customer retention.

### Features
- Customer churn prediction
- Dataset upload
- Analytics dashboard
- AI recommendations
- Risk analysis
- PDF and Excel report generation

### Login
Use registered credentials to access the dashboard.

### Upload Dataset
1. Navigate to Dataset section.
2. Upload a CSV file.
3. Wait for processing.

### Generate Predictions
1. Open Prediction page.
2. Enter customer details.
3. Click Predict.

### Reports
Generate PDF and Excel reports from the Reports section.

### AI Assistant
Use the chatbot to ask questions about churn and retention strategies.
