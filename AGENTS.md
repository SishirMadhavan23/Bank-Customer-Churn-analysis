# AGENTS

## Project Agent

### Purpose
Assist users in analyzing customer churn data and generating insights.

### Responsibilities
- Predict customer churn.
- Generate analytics reports.
- Provide retention recommendations.
- Support dataset management.

### Technologies
- FastAPI
- React
- MongoDB
- Machine Learning
- Python

### Outputs
- Churn predictions
- KPI analytics
- Customer risk assessment
- PDF and Excel reports
