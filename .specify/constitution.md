# Project Constitution

## Purpose

Bank Customer Churn Analysis is a platform designed to predict customer churn using machine learning and provide actionable business insights.

## Principles

1. Code must be documented.
2. All features must have specifications before implementation.
3. Security and privacy must be maintained.
4. Tests should accompany new features.
5. Changes should be reviewed through merge requests.

## Quality Standards

* Follow PEP8 guidelines.
* Maintain test coverage above 70%.
* Use automated linting and security checks.

## Documentation

All major features must include specifications and user documentation.
